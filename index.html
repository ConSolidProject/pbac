Ontology Specification Draft

Abstract

The PBAC vocabulary (Pattern-Based Access Control) is developed in context of the ConSolid project for heterogeneous data management on the Web. It complements existing access control mechanisms used in the Solid Ecosystem, by establishing pattern-based rules for both the clients and the requested resources.
